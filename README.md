# Zipa

A Terminal Based app for compress and decompress zipfiles

## Features

- [x] Compress
- [x] Decompress
- [ ] View content inside

## Run Locally

- Clone the project

```bash
  git clone git@github.com:CaioCDJ/Zipa.git
```

or

```bash
  git clone https://github.com/CaioCDJ/Zipa.git
```

- Go to the project directory

```bash
  cd Zipa
```

- Restore the project

```bash
  dotnet restore
```

- Running Zipa

```bash
  dotnet run
```
